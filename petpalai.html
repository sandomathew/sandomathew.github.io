PetPalAI

GenAI-powered Pet Management Platform

Centralizing pet health, food, and care with AI-driven insights.

Overview

PetPalAI is a scalable, modular Django-based GenAI web application designed for pet owners to manage all pet-related information in one place. It leverages LLMs for food label extraction, integrates user/pet management, and provides AI-powered recommendations.

Tech Stack

🐍 Python / Django
🧠 Ollama LLM
📊 Pandas, NumPy
🗄 SQLite → scalable to Postgres
🌐 HTML/CSS/JS Frontend

Key Features

🍖 Food Analysis – upload labels → AI extracts nutrition info.
🤖 AI Agents (in progress) – multi-step workflows like “Set up my new pet account” via natural language.
🐾 Pet Profiles (in progress) – medical history, vaccinations, appointments.
📊 Reports (future) – health & diet trends.
